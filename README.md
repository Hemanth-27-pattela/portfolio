# Hemanth Pattela — Personal Portfolio

A clean, responsive personal portfolio website built from scratch using HTML, CSS, and JavaScript.

## Live Demo
[hemanth-27-pattela.github.io/portfolio](https://hemanth-27-pattela.github.io/portfolio)

## Features
- Typing animation on hero section
- Dark / Light mode toggle
- Smooth scroll navigation
- Responsive project cards
- Skills section
- Contact section with LinkedIn link

## Built With
- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript

## Sections
- About — intro with typing animation
- Skills — tech stack pills
- Projects — cards with GitHub links
- Contact — email, phone, LinkedIn

## Projects Showcased
- Bus Route Management System (C++, DSA)
- Huffman Text Compression (C++, Algorithms)
- Student Record Management System (C, File Handling)
